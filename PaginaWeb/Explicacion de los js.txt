RELOJ.JS

En el js de mostrar la hora lo primero que hice fue crear la función mostrarHoraExacta. Lo que hace la función al principio es crear una variable llamada fecha que saca el momento exacto actual con el new Date(). Después, creé tres variables (horas, minutos y segundos) para separar y guardar cada dato por su cuenta usando los comandos de JavaScript correspondientes.

Luego vienen los if. Como a veces los números salen con un solo dígito, puse tres condiciones separadas si las horas, los minutos o los segundos son menores que 10, les sumamos un 0 de texto adelante para que visualmente quede bien.

Después, creé la variable relojcontador para juntar las horas, los minutos y los segundos separados por dos puntos, formando el texto del reloj.

Al final, buscamos en la página el elemento con el id reloj y lo guardamos en la variable Reloj. Si existe y lo encuentra, con el textContent cambiamos su texto para que muestre Hora actual: y la hora actual.

Para terminar todo fuera de la función primero llamo a mostrarHoraExacta() una vez para que la hora aparezca nada más cargar la página, y luego uso el setInterval para decirle que repita esa misma función cada 1000 milisegundos es decir 1 segundo así el reloj se vaya actualizando solo todo el tiempo.
